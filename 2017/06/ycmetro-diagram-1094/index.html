宜昌城市轨道交通一期建设规划线路示意图

宜昌市城市轨道交通一期建设规划（2017~2023）共包含3条线路，全长72.2km，其中地下线长约49.4km，高架线长约22.8km。
全线共设车站37座（换乘站不重复计算），其中地下车站29座，高架车站8座。总投资约为431.12亿元，工期6年。

// 6月23日更新
1号线一期始于葛洲坝商贸大厦附近的葛洲坝站，经夷陵大道、桔城路、城东大道、汉宜路至三峡职教园附近的土门站。魏家畈、土门为高架车站，其他车站均在地下。
2号线一期始于均瑶大厦附近的夷陵广场站，下穿东山后经城东大道、东站二路、三峡快速路、机场路、迎宾大道、猇亭大道至桃子冲村附近的猇亭站。张家湾、先锋路、叶家湾、猇亭为高架车站，其余车站为地下车站。
3号线一期始于岜峪坪村的岜峪坪站，经发展大道、东山隧道、西陵一路至均瑶大厦附近的夷陵广场站，与1、2号线形成换乘。岜峪坪、南村坪为高架车站，其余车站均为地下车站。
